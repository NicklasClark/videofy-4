== Description ==
Videofy is Modern Designed Vlogging WordPress Theme. Videofy is a Child Theme of VideoStories.

Our Support: http://prowptheme.com/support/

== Installation ==

1. In your WordPress Admin Panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the videofy.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
